#### 接收数据
* url:/api/receive_data
* 请求方法：POST
* 请求参数：表单
{
    protocal:string;协议
    port:Integer;端口
    catalogue:string;是来自哪个应用的流量
}
* 返回类型：JSON
*  返回格式：
{
    success:boolean;
    msg:string;
}
